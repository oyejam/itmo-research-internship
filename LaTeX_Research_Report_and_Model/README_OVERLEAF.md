# Overleaf Upload Instructions

This folder contains two LaTeX documents:

1. `reverse_engineered_mathematical_model.tex`
   - A standalone mathematical model for the reverse-engineered MPC UAV landing simulation.

2. `520194_Oyetunde_Jamiu_Jamiu_Research_Internship_Revised.tex`
   - A revised professional LaTeX version of the internship report, integrating the reverse-engineered paper's UAV/USV/MPC mathematical model.

## How to use in Overleaf

1. Create a new blank project in Overleaf.
2. Upload the `.tex` file you want to compile.
3. Set the compiler to `pdfLaTeX`.
4. Click `Recompile`.

No external image files are required. The documents use standard LaTeX packages available on Overleaf.

## Recommended submission workflow

First compile:

`reverse_engineered_mathematical_model.tex`

Then compile:

`520194_Oyetunde_Jamiu_Jamiu_Research_Internship_Revised.tex`

The first document may be used as a mathematical appendix or supporting material. The second document is the revised research report, which is still under review pending your verification of the code implementation, results, and generated figures, and their consistency with the original paper. After this review, a full manuscript can be prepared for submission to the ISU portal or as otherwise directed.
